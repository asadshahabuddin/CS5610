Registration

Description

MEAN is used to create an account on my project website. Authentication information persists in a MongoDB database and you will be logged in after registering successfully. Extense console logging has also been done as an additional feature for developers. Without registration or stealing someone else's information, you cannot log into the web application. What follows after signing in a scope for future experiments.

Click on Sign In link on the top right portion of my project website. On arriving at the form location, click on the Register a new account checkbox to register. Enter the required information, click on Register and you're good to go. Certain fields are mandatory (they will be indicated when you violate rules) and password entries must coincide.

Account Registration

A HTTP POST request is issued by the AngularJS registration function on user request. The aforementioned request is handled by the Node.js server file which adds user information document to a MongoDB collection. Future login requests from this user are validated against this collection. Email ID and password must be entered and the two passwords must match.

app.post("/api/register", function(req, res)
{
var user = new UserModel(req.body);
user.save(function(err, doc)
{
req.login(user, function(err)
{
if(err)
{
return next(err);
}
res.json(
{
username : user.username,
firstName: user.firstName,
lastName : user.lastName,
city : user.city
});
});
});
});
