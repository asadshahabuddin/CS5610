CSS Image Swap

Objective

CSS has been used to implement image swap. The image area shows a picture in its initial state, which swaps to something else on a mouse hover event. When the mouse moves out of the image area, the initial picture comes into place once again.

Hover your mouse over the image.

Documentation

Image alternate on a mouse hover event

.imageArea
{
background-image: url("darwin1.jpg");
height: 300px;
width: 400px;
}

.imageArea:hover
{
background-image: url("darwin2.jpg");
height: 320px;
width: 320px;
}
