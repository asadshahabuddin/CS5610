Daily Horoscopes

Description

Accordian menus are used to create a daily horoscope page in the form of a navigation list. Clicking on a horoscope title opens up the description which is a complete horoscope summary for the day. The accordian widget results in lucid collapsible components. The first element is open by default. Clicking on another title opens up the corresponding description and closes the other, whereas clicking on the same title again closes all the components. Panels are used to create the overall interface.

Documentation

Accordian and Collapsible Components

Panels and panel groups are used to encapsulate collapsible components, with the 'panel-group' id set to 'accordion' to fetch the suitable Bootstrap component. Collapsible data toggles are used to ensure that clicking on a navigation title toggles the collapsible property of that accordion. A division tag set to 'in' class remains open by default. Clicking on another item opens it up and closes others.

<div class="panel-group" id="accordion">
<div class="panel panel-default">
<div class="panel-heading">
<h4 class="panel-title">
<a data-toggle="collapse" data-parent="#accordion" href="#aries">Aries</a>
</h4>
</div>
<div id="aries" class="panel-collapse collapse in">
<div class="panel-body">
<p>
...
</p>
</div>
</div>
</div>
</div>
