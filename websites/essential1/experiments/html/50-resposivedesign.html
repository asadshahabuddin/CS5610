Responsive Design

Description

My project website consists entirely of hand-written CSS and does not use Bootstrap. Because of this, horizontal scroll bars would appear until last week. I carefully tuned my CSS to make sure that no page has a horizontal scroll bar on it even when resized to the smallest possible size. This has been achieved by replacing static width assignment for various elements.

Media queries

All width assignments use percentages instead of pixel counts. In addition to those, media queries have been used to resize elements when the screen size goes below a certain number of pixels. This creates a partly responsive design. A caveat is that the navigation bar does not resize in the best possible way, which remains a scope for future improvement.

@media screen and (max-width: 500px)
{
.asFormElem
{
font-size: 14px;
}
}

@media screen and (max-width: 500px)
{
.asH1
{
font-size: 3em;
}
}

@media screen and (max-width: 420px)
{
.asH1
{
font-size: 2em;
}
}

.asImgFrame
{
background-size: cover;
float: left;
height: 150px;
margin-right: 15px;
margin-bottom: 15px;
width: 29%;
}
