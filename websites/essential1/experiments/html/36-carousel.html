Description

This week's experiments are based off my final project work and highlight different aspects of it which have been refined and developed during the course of past few days, as well as issues encountered and results achieved. All of my experiments navigate to the same URL and highlight various parts of my project so far.

The first experiment exhibits a CSS carousel not based on Bootstrap. I had help from an independent library called Owl Carousel.

Documentation

CSS Carousel

Fine-grained CSS has been written to make sure the carousel works across a variety of browsers. Specific properties are coded for webkit-based and other kinds of browsers. Transition effects and timings have been made possible by the use of JavaScript.

/* CSS */
.asCarouselItem
{
display: block;
font-size: 4em;
height: 480px;
width: 100%;

-webkit-border-radius: 3px;
-moz-border-radius: 3px;
border-radius: 3px;
}

.asSpanOverImg
{
background: rgb(0, 0, 0); /* Fallback color */
background: rgba(0, 0, 0, 0.7);
color: white;
padding: 10px;
}

/* JavaScript */
$(document).ready(function()
{
$(".asCarouselDiv").owlCarousel(
{
autoPlay : 3000,
stopOnHover : true,
navigation : false,
paginationSpeed : 1000,
goToFirstSpeed : 2000,
singleItem : true,
autoHeight : true,
transitionStyle :"fade"
});
});
