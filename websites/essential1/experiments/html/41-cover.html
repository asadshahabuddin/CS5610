Profile

Description

When a user signs into my website, they are navigated to their home page with a cover picture. You can click on the Change cover button to change it. The selection list is pre-populated and you may select an image from it. The change sticks between sessions during the application's lifetime.

Register an account with the website and you should see your homepage with a default cover. Change it as many times as you want and have it saved for your profile.

Node.js server hosted at OpenShift

Documentation

Cover Picture

A modal dialog is used to present a list of pictures from which you can pick one to have it as a cover. Upon selection, press ESC to head back to your original window. The cover picture is also overlayed with your initials to make it look more stylized.

/* Display modal window */
$scope.coverModal = function()
{
document.getElementById("as-cover-modal").showModal();
};

/* Set cover picture */
$scope.cover = function()
{
$http.get("/api/user/" + $rootScope.currentUser._id + "/cover")
.success(function(res)
{
document.getElementById("as-cover-div").style.backgroundImage = "url(" + res + ")";
});
}

/* Change cover picture */
$scope.changeCover = function(file)
{
document.getElementById("as-cover-div").style.backgroundImage = "url(" + file + ")";
file = file.replace("image/", "image-");
file = file.replace(".", "-");
$http.put("/api/user/" + $rootScope.currentUser._id + "/cover/" + file)
.success(function(res)
{
$scope.trace("Changed your cover picture");
});
};
