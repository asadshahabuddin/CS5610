Activity Feed

Description

Once you sign in to your profile, you can perform a number of activities such as change your cover, navigate to the News section, query weather conditions for any city (inside the News page), search for a book, add it to your favorites and view bookmarks. Whenever you perform any of these, your activity feed shows you a record of past actions. It changes as you go.

Perform any or all of the actions mentioned and navigate to your profile page (the link on navigation bar with your name on it). You should see a record of your past actions beneath the cover.

Documentation

Logging

Two functions - trace() and feed() - are created to log recent events and display them on one's profile page respectively. They make changes to the database and everything is stored persistently. The list keeps growing and my future plans include shortening it or displaying it in a way that doesn't lead to infinite scrolling over time.

/* Log user activities */
$scope.trace = function(msg)
{
$http.put("/api/user/" + $rootScope.currentUser._id + "/trace/" + msg)
.success(function(res)
{
$scope.feed();
});
};

/* Fetch trace */
$scope.feed = function()
{
$http.get("/api/user/" + $rootScope.currentUser._id + "/trace")
.success(function(res)
{
if(res != null)
{
$scope.activities = res.activity;
}
});
};
