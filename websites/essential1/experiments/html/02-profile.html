HTML Profile Page

Objective

A formatted profile page in HTML and CSS. It explores the CSS syntax and selectors to build up on the profile page. Formatting properties have been defined for the entire body as well as individual tags and or/classes. In addition to this, a horizontal navigation bar that sits atop the page has been included.
