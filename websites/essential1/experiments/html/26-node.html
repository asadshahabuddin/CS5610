Node.js Server

A server.js file is hosted on OpenShift which makes it possible for me to query author data. As mentioned earlier, the file will run locally to produce equivalent results as CORS is enabled for this particular query. app.get(...) handles GET requests for a specified URL.

/* Author */
var author = {
first: "Asad",
last : "Shahabuddin"
};

app.get("/api/author", function(req, res)
{
res.setHeader("Access-Control-Allow-Origin", "*");
res.json(author);
});
