News

Description

I have used The New York Times API and the Weather API to create a profile page. Both APIs return JSON data which is parsed to create a simple and intuitive interface. Recent headlines are displayed with newest on top. Clicking on a headline will take you to the full story on The New York Times website. Weather information is presented in its own card on the right.

Sign in using your previously created credentials and you should see a homepage with news and weather information. Your name will appear on the navigation bar.

Node.js server hosted at OpenShift

Top Stories

A HTTP GET request is utilized to query Times Developer Network. The result is parsed and presented as a list of paragraphs. This interface scales well on all screen sizes as verified on a desktop and an iPhone screen. The active user's name is determined from the root scope, which stores user details and relavant error messages, if any.

/* Fetch headlines using The New York Times Top Stories API */
$http.get("http://api.nytimes.com/svc/topstories/v1/home.json?api-key=6b50b9dd8f1647a9456befedcda60f26:6:71641947")
.success(function(res)
{
console.log("%cHeadlines>",
"font-family: Courier New; font-weight: bold;");
console.log(res);
$scope.headlines = res.results;
});
